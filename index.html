Pure JS Arkanoid v0.8.6.1

The game is pretty simple. You have to destroy as many bricks as you can.

To move a pad use left/right (A/D) buttons. In order to stop the game just push "Esc".

So what are you waiting for? Type your name, a color and click "PLAY!" button!

Check description here

You will earn 25 points for each brick hit as well as 750 points for each levelUP.

Some bricks include buffs and some debuffs. Avoid red blocks catching and try to catch green, as you will get debuff for each missed green block. By default, you have 3 live but you can catch more in green blocks as well as miss.

The result will be saved in your browser's local storage, so "Reset the results" feature means the storage clearing.
